<code>

</code>

## 3.Set class and move amount
<code>

<div class="jqsprite" data-y="amount">
<div class="jqsprite" data-x="amount" data-y="amount">

</code>

If you want to set first postion use css(top%left).
jqsprite class is fixed positioned.
